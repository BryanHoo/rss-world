# AptFlow Logo 设计（Flow-A）

## 背景
- 需要完全替换当前 RSS 语义图标。
- 使用场景以侧栏小图标（约 `18x18`）和 favicon 为主，优先保证小尺寸识别。

## 目标
- 设计全新 `A` 字母品牌标识，不保留 RSS 弧线/信号点元素。
- 视觉方向：现代、流动、简洁。
- 输出图标版与字标版两套资产。

## 方案对比
1. `Flow-A`（选中）
- 两条流线围合成字母 `A`，中心保留三角负空间。
- 优点：在 `16x16` 下仍具备高识别度；与 `AptFlow` 的 Flow 语义一致。

2. `Mono-A`
- 单笔几何 `A`，切角极简。
- 优点：实现快、技术感强；缺点：品牌独特性一般。

3. `Frame-A`
- 圆角框内嵌 `A`。
- 优点：favicon 稳定；缺点：灵动感较弱。

## 定稿规范
### 图标结构
- 画布：`128 x 128`。
- `A` 主体占画布约 `74%`。
- 顶部为轻微圆切角，避免锐角在小尺寸发糊。
- 中间以三角负空间形成 `A` 的中段识别。
- 禁止出现 RSS 弧线、同心圆、信号点等旧语义。

### 配色
- 主梯度：`#0F172A -> #2563EB -> #22D3EE`，沿流向渐变。
- 默认透明背景。
- 仅在 app icon 需要底板时可使用深色圆角方块底。

### 字标规则
- 组合关系：图标在左，`AptFlow` 在右。
- 图标高度约为字标大写高度的 `1.15x`。
- 字标选用现代无衬线，字距略收紧。
- 必须提供单色深色版和单色白色版 fallback。

- `public/rss-world-logo.svg`（替换为新 `Flow-A` 图标）

## 验收标准
- `16x16` 下可明确识别为 `A`。
- 侧栏 `18x18` 下轮廓清晰，不粘连。
- 图标版与字标版视觉语言一致。
- 所有导出资产不包含 RSS 相关图形语义。
